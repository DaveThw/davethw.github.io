## DMX address
1-255 via dip switches - doesn't work with a base address of 256 or above!

## ch. 1
| Value   | Colour           |
|:-------:|:-----------------|
| 0-8%    | open white       |
| 9-16%   | pale blue        |
| 17-24%  | green            |
| 25-32%  | yellow           |
| 33-40%  | dark blue        |
| 41-49%  | purple           |
| 50%     | open white       |
| 51-100% | colour cycle, fast-slow (order as above) |
